Speed up your website. Optimize your JPEG, PNG, and WebP images automatically with babypng.

== Description ==

Make your website faster by optimizing your JPEG, PNG, and WebP images. This plugin automatically optimizes all your images by integrating with the popular image compression services.

= Features =

* Automatically optimize new images on upload.
* Advanced background optimization to speed up your workflow.

= How does it work? =

After you upload an image to your WordPress site, each resized image is uploaded to the baypng service. Your image is analyzed to apply the best possible optimization. Based on the content of your image an optimal strategy is chosen. The result is sent back to your WordPress site and will replace the original image with one smaller in size. On average JPEG images are compressed by 40-60% and PNG images by 50-80% without visible loss in quality. Your website will load faster for your visitors, and youâ€™ll save storage space and bandwidth! During the uploading process of images to the media library, the BABYPNG plugin simultaneously sends the images to the babypng server using babyPNG API's and will get optimze image in API response [babypng.com](https://babypng.com) (BabyPNg API's). 

= Getting started =

Install this plugin and follow the instructions to set up your account. 



== Installation ==

= From your WordPress dashboard =

1. Visit *Plugins > Add New*.
2. Search for 'babypng' and press the 'Install Now' button for the plugin named 'babypng - JPEG, PNG & WebP image compression' by 'babypng'.
3. Activate the plugin from your *Plugins* page.
